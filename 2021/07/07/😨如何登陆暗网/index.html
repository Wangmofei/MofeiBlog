首先介绍一下大名鼎鼎的工具“洋葱头Tor”

Tor Project, Inc. 是一家位于西雅图的 501(c)(3) 研究教育非营利组织，由计算机科学家Roger Dingledine、Nick Mathewson和其他五人创立。Tor 项目主要负责维护Tor “Tor（匿名网络）

Tor 网络就是我们所称的“暗网”。

准备打开新世界的大门

安装Tor Browser

  1. 如果有 TORY 插件，需要删除并重新安装最新版本，设置 lazyload.isSPA: true。
  2. 2.x 版本的 css 和 js 不适用于 3.x 版本，如果使用了 use_cdn: true 则需要删除。
  3. 2.x 版本的 fancybox 标签在 3.x 版本中被重命名为 gallery 。
  4. 2.x 版本的置顶 top: true 改为了 pin: true，并且同样适用于 layout: page 的页面。
  5. 如果使用了 TORY-offline 插件，建议卸载，3.0 版本默认开启了 pac 服务。

下载源码查看文档

使用方法

  1. 如果你有VPN但不想使用VPN作为Tor的代理，想直接使用网桥：你可以进入https://bridges.torproject.org/bridges 获取最新的共享网桥。

  2. 你仍然可以使用其他可用的网桥来保证你连接上Tor服务。如果你想自己搭建网桥：去国外或者香港的云服务器提供商购买一个vps并参照网络上的资料搭建即可使用。

  3. 成功连接上Tor后，那么恭喜你，成功进入了Tor网络，你可以进行彻底的进行无痕迹浏览了。
