🥩超级无敌美味私房菜谱

|字数总计:3k|阅读时长:9分钟|阅读量:

🍔大王的私房菜谱🍲
🥟🍗🍖🍹🍜🍢🍡

红烧排骨

主料：排骨800g
配料：八角2个、桂皮1个、香叶1片、花椒1把、大葱1节、姜片4片
辅料；盐、生抽、老抽、耗油、料酒、冰糖

  1. 首先洗净新鲜排骨，放入锅中焯水去血沫。
  2. 准备葱姜、花椒、香叶、桂皮、八角、冰糖，加入油锅翻炒至冰糖融化
  3. 加入排骨翻炒上糖色至金黄
  4. 加入生抽、老抽、料酒翻炒半分钟
  5. 加水至没过排骨，慢炖1小时，加入耗油调味
  6. 大火收汤至汤汁浓稠，起锅盛盘，点缀小葱

鲍鱼鹌鹑红烧肉

主料：五花肉1000g、鲍鱼6个、鹌鹑蛋200g
配料：八角4个、桂皮2个、香叶2片、大葱半根、姜5片、话梅5个
辅料；生抽、耗油、料酒、冰糖

  1. 首先洗净新鲜五花肉，放入锅中焯水去血沫。
  2. 半热油锅翻炒冰糖至融化,加入排骨翻炒上糖色至金黄，盛出备用
  3. 准备葱姜、香叶、桂皮、八角、冰糖，加入油锅翻炒至冰糖融化
  4. 加水至没过五花肉，慢炖30分钟后加入鹌鹑蛋，慢炖20分钟，加入鲍鱼和话梅，慢炖15分钟
  5. 大火收汤至汤汁浓稠，起锅盛盘，点缀小葱

糖醋里脊

主料：里脊肉500g、玉米淀粉20g、番茄酱100mg、白芝麻10g
配料：无
辅料；料酒、白糖、白醋、盐

  1. 首先洗净新鲜里脊肉，切成细长条备用
  2. 碗中倒入盐、料酒腌制20分钟后，加入玉米淀粉包裹均匀
  3. 半热油锅逐条放入里脊条,炸至六分熟
  4. 滚烫油锅下入全部里脊，复炸至金黄捞出备用
  5. 准备白醋、白糖、番茄酱，倒入锅中翻炒30秒
  6. 加入金黄里脊肉翻炒上色，起锅盛盘

三杯鸡

主料：鸡腿肉500g
配料：蒜半头、姜片4片（这入小碗）
辅料；盐、老抽、百里香、耗油、料酒、白砂糖

  1. 首先洗净新鲜鸡腿肉，去骨剔肉。
  2. 准备姜片、蒜末，用香油鞭炒香
  3. 倒入鸡腿肉翻炒至肉质偏干
  4. 加入少许耗油、大量料酒、白砂糖翻炒
  5. 加入老抽上色，点缀一些百里香，起锅盛盘

牛腩萝卜

主料：牛腩500g、白萝卜300g（切滚刀块）
配料：八角2个、桂皮1个、花椒1把、蒜半头、姜片4片（这入小碗）
辅料: 盐、耗油、冰糖3块、黑胡椒、白胡椒、柱侯酱

  1. 首先将牛腩洗净备用，将小碗中的配料入油翻炒
  2. 倒入牛腩翻炒至肉质偏干
  3. 加入冰糖 、黑胡椒、白胡椒、耗油、柱侯酱、耗油 、盐，继续翻炒
  4. 加入高压锅慢炖半小时 最后加入萝卜炖10分钟，加入一点鸡精
  5. 煮米粉，起牛腩锅，盛盘加入米粉

浓香炖带鱼

主料：鲜带鱼1000g、鸡蛋一颗
配料：八角3个、桂皮2个、花椒1把、大葱2节、蒜1头、姜片8片、香叶1片
辅料: 白酒食醋、生抽、老抽、蚝油、鸡精、盐、白砂糖

  1. 将新鲜带鱼洗净，切成段儿后擦干水份彻底晾干
  2. 锅内放油将带鱼煎至两面金黄（加鸡蛋液），盛出带鱼备用
  3. 锅内下底油，加入葱姜蒜、八角、花椒、桂皮、香叶、大料煸炒
  4. 加水至没过带鱼，倒入白酒、食醋、生抽、老抽、蚝油、白糖
  5. 小火焖炖20分钟后，开锅收汤，将汤汁收到浓稠

王记油泼面

主料：新鲜切面300g
配料：大葱2节、蒜1头、十三香10g、辣椒面10g、白芝麻10g
辅料: 生抽、鸡精、盐、芝麻

  1. 将新鲜面条煮熟置于碗中
  2. 面上铺满蒜末、葱花、鸡精、白芝麻、十三香、辣椒面
  3. 小锅烧热油，浇到面上，点些酱油

番茄炒鸡蛋

主料：西红柿500g、鲜鸡蛋4颗
配料：大葱1节
辅料: 味极鲜、盐、白芝麻10g

  1. 将西红柿洗净，切成滚刀块后擦干水份彻底晾干
  2. 锅内放入底油，将鸡蛋糊倒入锅中翻炒，盛出备用
  3. 锅内放入底油，将西红柿倒入锅中，加入番茄酱翻炒片刻（可加糖）
  4. 加入炒鸡蛋翻炒片刻，关火放葱提味

可乐鸡翅

主料：鲜鸡翅1000g
配料：大葱2节、蒜8瓣、姜片8片、可乐1瓶
辅料: 生抽、蚝油

  1. 将新鲜鸡翅洗净，加入葱姜，煮开5分钟
  2. 锅内放油将鸡翅煎至表面金黄，盛出鸡翅备用
  3. 锅内下底油，加入葱姜蒜、鸡翅煸炒
  4. 倒入生抽、蚝油、盐调味儿，煸炒5分钟
  5. 倒入可乐至没过鸡翅，盖盖闷炖1小时
  6. 开锅收汤，将汤汁收到浓稠

白灼基围虾

主料：鲜虾500g
配料：大葱2节、姜片8片、花椒1把、柠檬1片、小辣椒1个
辅料: 生抽

  1. 将鲜虾洗净去虾线，加入葱姜，煮开3分钟
  2. 蘸料：柠檬切片，小辣椒切片，加入生抽拌匀

清蒸海鲈鱼

主料：新鲜海鲈鱼500g
配料：大葱2节、姜片8片、花椒1把、柠檬1片、小米辣椒4个
辅料: 生抽

  1. 将鲜鱼洗净，加入葱姜，上锅蒸10分钟
  2. 锅内放入底油，烧至冒青烟，加入生抽烹制出霍气
  3. 盖锅盖静置至油温七分热，霍气溶解，加入切好的葱丝
  4. 将酱汁淋在蒸鱼，出锅盛盘

秘制川味鸡丝凉面

主料：切面500g、鸡丝50克
配料：姜片8片、花椒1把、大蒜1头
辅料: 生抽、蚝油、白糖、辣椒油、醋

  1. 将切面煮熟过凉水，加入香油拌匀放凉
  2. 将锅烧热，加入花椒烤至焦香，用擀面杖捻碎备用
  3. 按生抽、醋、耗油比例2：1：0.5配置料汁，其他配料依喜好添加
  4. 加入鸡丝，将酱汁淋在面上

葱爆羊肉

主料：羊肉片500g、洋葱1个
配料：大葱4节、姜片8片、花椒1把
辅料: 玉米淀粉、生抽、料酒、蚝油、剁辣椒酱

  1. 将羊肉片解冻、挂玉米淀粉糊拌匀
  2. 锅内下底油烧热，将葱姜末和花椒炒香炒香
  3. 加入羊肉片翻炒片刻烹入料酒生抽炒熟备用
  4. 锅内下底油烧热，加入洋葱片和生抽烹炒至洋葱变软
  5. 倒入备好羊肉，加入蚝油、剁辣椒翻炒片刻后出锅

冰镇梅渍圣女果

主料：圣女果300g、话梅50g、柠檬5片、雪碧1听
配料：无
辅料: 无

  1. 将圣女果去根，尾部切十字刀，焯水10秒后去皮备用
  2. 干净密封容器内加入去柠檬片、话梅、去皮圣女果和雪碧
  3. 将容器放入冰箱密封腌制一晚后可食用

泰式鲜虾蔬菜沙拉

主料：鲜虾300g、洋葱1/4个、香菜、柠檬3片
配料：南姜片2片、小米辣椒3个、香茅草粉、九层塔、冬阴功酱
辅料: 红烧汁、椰汁

  1. 将鲜虾洗净去虾线，加入葱姜，煮开3分钟后备用
  2. 将洋葱切丁，香菜、南姜和小米辣椒切末，柠檬片挤出汁
  3. 加入香茅草粉、九层塔、冬阴功酱、椰汁
  4. 加入鲜虾和少量无盐红烧汁拌匀

十全甜品滋补汤

主料：银耳3块、红枣10颗、龙眼10颗、榛子10颗、核桃5颗、特技枸杞20g、山楂条5条
配料：姜片8片、阿胶红糖30g、良食记醪糟500g、鸡蛋1个
辅料: 矿泉水

  1. 将龙眼、榛子、核桃去壳洗净，银耳泡发备用
  2. 砂锅内倒入矿泉水，冷水加入红枣、核桃、榛子、姜片、阿胶红糖炖煮10分钟
  3. 加入银耳、龙眼、醪糟、山楂条、再炖煮10分钟
  4. 鸡蛋打散成蛋液，缓缓倒入沸腾汤汁成蛋絮
  5. 关火前加入枸杞，5分钟后即可食用

土豆咖喱饭（待更新）

主料：牛肉300g、糯米土豆2个、洋葱2个、胡萝卜1个
配料：料酒
辅料: 好侍黑盒辣咖喱一盒、椰汁1/2份、牛奶1盒子

  1. 牛肉、胡萝卜、洋葱、土豆切块备用
  2. 锅中加水烧热加入牛肉焯水，料酒去腥撇沫备用
  3. 油锅烧热加入洋葱炒香
  4. 加入咖喱块与牛肉、胡萝卜、土豆翻炒片刻加水至没过食材
  5. 加入椰汁、牛奶慢炖20分钟即可

无名贝果

主料：鸡蛋1个、贝果1个、牛油果1/2个、
配料：Wartburg Cream Cheese、海盐、黑芝麻
辅料: 橄榄油

  1. 平底锅热油煎溏心蛋，贝果用橄榄油上锅烤热，牛油果切片备用
  2. 贝果涂干酪，夹牛油果和溏心蛋
  3. 撒上海盐和黑芝麻即可

虫草人参鸡汤

主料：散养走地老母鸡半只、人参3根、虫草一把、黄芪4个、红枣4个
配料：大葱4节、姜片8片、枸杞一把
辅料: 无

  1. 鸡肉焯水和食材洗净备用
  2. 锅中加矿泉水，加入所有食材小火慢炖3小时
  3. 起锅前加入枸杞，小碗盛出加盐调味即可

美式热狗

主料：宜家牛肉肠1根、盒马热狗坯1个
配料：Master芥辣酱、Hertz番茄酱、酸黄瓜酱
辅料: 无

  1. 面包加热30秒，肉肠加热1分钟
  2. 面包夹肠，涂Master芥辣酱、Hertz番茄酱、酸黄瓜酱即可

香煎西冷牛排

主料：山姆西冷牛排500g、芦笋100g、圣女果5个、蒜头3个、迷迭香2根
配料：黄油15g、黑胡椒3g、海盐3g
辅料: 橄榄油

  1. 将牛排擦干后双面涂橄榄油，涂抹海盐和黑胡椒腌制10分钟。
  2. 锅内放油烧热放牛排，单面煎50秒至牛排两面金黄后封边。
  3. 捞出牛排放黄油，烧热后放入蒜头、圣女果、迷迭香、芦笋，翻炒片刻淋在牛排上。
  4. 煎一个鸡蛋做装饰，出锅装盘。
